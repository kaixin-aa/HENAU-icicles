# 数据结构模拟卷

> **项目结构演示，不是真实历年试卷。**
>
> 本卷全部题目由本项目重新编写，仅用于演示公开课程仓库的资料格式。建议用时 60 分钟。

## 一、复杂度分析

分析下面伪代码的时间复杂度，并说明理由：

```text
for i = 1 to n:
    j = 1
    while j <= n:
        visit(i, j)
        j = j * 2
```

## 二、栈操作

一个空栈依次执行：

```text
push(3), push(5), pop(), push(2), peek()
```

写出 `pop()` 的返回值、`peek()` 的返回值和最后从栈底到栈顶的元素序列。

## 三、二叉树遍历

给定二叉树：

```text
        A
       / \
      B   C
     / \   \
    D   E   F
```

分别写出前序、中序和后序遍历序列。

## 四、二叉搜索树

从空树开始依次插入 `40, 20, 60, 10, 30, 50, 70`。

1. 写出所得树的中序遍历；
2. 删除节点 `20`，若规定使用其中序后继替代它，说明替代节点并画出删除后的局部结构。

## 五、图遍历

无向图按字母顺序访问邻接点，邻接关系如下：

```text
A: B, C
B: A, D, E
C: A, F
D: B
E: B, F
F: C, E
```

从 `A` 出发，写出广度优先遍历序列和深度优先遍历序列。遍历过程中已经访问的顶点不再重复进入队列或递归。

完成后请对照 [学生解答](./学生解答.md)。
