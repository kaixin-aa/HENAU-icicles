# 数据结构

> 计算机专业课程示例页。当前资料均为本项目原创演示内容，不对应某位教师或某次真实考试。

数据结构关注如何组织数据，以及不同组织方式如何影响操作效率。学习重点不只是记住数组、链表、树和图的定义，而是根据操作需求选择结构、描述算法，并分析时间和空间复杂度。

## 建议顺序

1. 渐进复杂度与基本算法分析；
2. 线性表、链表、栈和队列；
3. 字符串、递归与查找；
4. 二叉树、堆和搜索树；
5. 图的表示、遍历和最短路；
6. 排序、哈希及结构选择。
