# 文件说明

本目录收录线性代数课程相关资料，仅供学习交流使用。资料来源于课程资料整理和网络公开资料汇总。

## 目录结构

- `考试/`：按学年和学期整理的历年试卷及参考答案。
- `练习与题库/`：未明确归入具体考试学期的练习、题库和补充题。
- `知识点/`：图片形式的知识点笔记。

## 命名说明

原始文件名基本保留，便于追溯资料来源和对应关系。形如 `11-12-1-A.pdf` 的文件整理到 `2011-2012-1/`，其中最后一位数字表示学期；形如 `12年线性代数.pdf` 的文件整理到对应学年的 `未标明学期/` 目录。

## 使用提示

除明确标注的参考答案外，其余答案均为非标准答案，仅供复习核对。若发现文件命名、卷别、答案对应关系或内容归类存在问题，建议通过 Issue 或 Pull Request 纠正。
