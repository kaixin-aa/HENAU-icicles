# 线性代数

> 公共基础课程示例页。当前资料均为本项目原创演示内容，不代表河南农业大学具体教学安排。

线性代数研究线性方程组、矩阵、向量空间、线性变换和特征值等内容。学习时应把“计算步骤”和“结构含义”结合起来：行变换不仅是在算数，也是在识别方程之间的约束；特征值不仅是一道行列式题，也描述了线性变换保持方向的方式。

## 建议顺序

1. 矩阵运算与初等行变换；
2. 线性方程组、秩和可解性；
3. 向量组、线性相关、基与维数；
4. 行列式与可逆矩阵；
5. 特征值、特征向量与对角化；
6. 正交、投影和二次型。
