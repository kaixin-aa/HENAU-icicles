# 线性代数

> 公共基础课程资料。当前收录内容包括历年试卷、参考答案、练习题库和知识点图片笔记。

线性代数主要涉及矩阵、线性方程组、向量组、特征值与特征向量、相似对角化和二次型等内容。复习时建议把计算流程和概念判断结合起来，尤其注意矩阵初等变换、线性相关性、特征值性质与二次型正定性的常见题型。

## 复习建议

1. 先整理矩阵运算、行列式、逆矩阵和初等变换的基础计算。
2. 再集中练习线性方程组、向量组秩、线性相关与极大无关组。
3. 最后复习特征值、特征向量、相似对角化和二次型。
4. 做历年卷时建议把试卷和答案分开使用，先限时完成，再对照答案核查。
