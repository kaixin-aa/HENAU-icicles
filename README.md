# 河南农业大学课程攻略共享计划

> HENAU-icicles · 非官方、非营利的课程攻略与学习资料共享计划。

来到大学以后，我们常常需要在群聊、网盘和学长学姐之间反复寻找课程经验。很多资料在一届学生完成课程后便逐渐散失，而下一届又要重新摸索。这个项目希望把可以合法公开的学习方法、复习经验和原创练习整理成易于浏览、可以持续纠错的 GitHub 课程目录。

本仓库的组织方式参考了 [QSCTech/zju-icicles](https://github.com/QSCTech/zju-icicles) 的“每门课程一个目录、课程 README 写攻略、资料分类保存”思路；本项目不复制其课程资料或文案。

## 课程导航

| 课程 | 类型 | 当前内容 |
| --- | --- | --- |
| [线性代数](./线性代数/) | 公共基础课 | 历年试卷、参考答案、练习题库、知识点图片 |
| [数据结构](./数据结构/) | 计算机专业课 | 复习资料、练习与答案、历年试卷 |
| [军事理论](./军事理论/) | 公共基础课 | 练习题库、历年试卷、学生作答 |
| [中国近现代史纲要](./中国近现代史纲要/) | 公共基础课 | 2013—2025 年历年试卷、参考答案及评分标准 |

从本页进入课程后，可以继续点击具体资料；不需要下载整个仓库。

## 收录内容

- 课程学习顺序、复习建议和常见误区；
- 贡献者原创的笔记、练习题、模拟卷和解答；
- 已结束考核且具有明确公开权利的历年资料；
- 合法来源链接及必要的使用说明。

线性代数、数据结构、军事理论和中国近现代史纲要课程已收录复习资料、练习答案与已经结束考核的历年试卷。除明确标注的参考答案及评分标准外，其余答案均为非标准答案，仅供复习核对。

## 如何贡献
